**Stack:** HTML, CSS, JavaScript, Bootstrap (no frameworks)

## Run locally

1. Clone or copy the project folder.
2. Ensure files are in this structure:
   - `index.html`
   - `css/custom.css`
   - `js/main.js`
   - `assets/` (images)
3. Open `index.html` in your browser (double-click or run a local static server).
   - Recommended: use `npx http-server` or `python -m http.server 8000` from project root.

## Notes

- Replace placeholder images in `/assets/` (logo.png, hero.jpg, menu1.jpg,... gallery images).
- Update Google Maps iframe `src` with your place's embed code.
- The contact/reservation form uses front-end validation only (no backend).
- Optional: Add favicons, meta tags and real phone/email before publishing.
